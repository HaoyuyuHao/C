# **第一章课后习题**
## 1.1 查找资料，总结目前排名前10的编程语言各自的特点和主要应用领域
### 1.Python
- **特点**：语法简洁如英语、生态极其丰富、开发效率极高
- **主要应用领域**：人工智能（AI/ML）、数据科学、自动化脚本、后端开发
### 2.JavaScript
- **特点**：全栈通用、异步处理能力强、Web 标准语言
- **主要应用领域**：前端开发、Node.js 后端、跨平台移动应用、网页交互
### 3.Java
- **特点**：性能稳定、跨平台（JVM）、强大的企业级生态
- **主要应用领域**：企业级后端、安卓开发、金融系统、大数据（Hadoop）  
### 4.C++  
- **特点**：极致性能、硬件控制力强、语法复杂
- **主要应用领域**：游戏引擎（Unreal）、高频交易、系统级软件、图形处理
### 5.C#
- **特点**：微软生态核心、语法优雅、工具链（VS）强大
- **主要应用领域**：Unity 游戏开发、Windows 桌面应用、企业级 Web 服务
### 6.TypeScript
- **特点**：JavaScript 的超集、静态类型检查、适合大项目
- **主要应用领域**：大型 Web 前端项目、复杂后端架构（配合 NestJS 等）
### 7.Go (Golang)
- **特点**：并发模型卓越（Goroutine）、编译快、部署简单
- **主要应用领域**：云原生/容器化（Docker/K8s）、微服务、后端基础架构
### 8.SQL
- **特点**：专为数据设计、声明式语言、不可或缺
- **主要应用领域**：数据库管理、数据分析、商业智能（BI）
### 9.C
- **特点**：底层之王、内存效率极高、几乎无运行损耗
- **主要应用领域**：嵌入式系统、操作系统内核、驱动程序、IoT 设备
### 10.Rust
- **特点**：内存安全（无 GC）、极致性能、现代语法
- **主要应用领域**：系统编程、区块链（Web3）、高性能 WebAssembly 模块
## 1.3 查找资料，解释什么是图灵测试
询问者（C）： 一名人类裁判。  
受试者（A 与 B）： 一个是人类，另一个是计算机程序。  
规则： 询问者通过键盘和显示器（以避免视觉和声音干扰）与两者进行对话。如果询问者在经过长时间的交流后，无法分辨哪一个是人，哪一个是机器，那么这台机器就通过了图灵测试，被认为具有“智能”。

## 1.7 程序在计算机内部是如何运行的
简单来说，程序在计算机内部的运行可以看作是一场由 CPU（中央处理器） 指挥、内存（RAM） 配合的“极速翻译与执行”过程。
